About the Pathing Program

This web based program is created entirely with Html and JavaScript. It was designed to teach students new to algorithms about some of the most common tree / graph searching algorithms in computer science. The program is able to do this by providing a visual reference of the data being worked on as well as a red box signifying where actions are currently taking place. Along with a short description about each algorithm this program is designed to show how those algorithms work behind the scene to trace through a given set of data in order to find the target element.

The program visualizes all elements as colored squares. Unvisited elements are grey, visited are green, walls are black, starting point is yellow, endpoint is red, and the path is blue. Each pathing algorithm has been broken down to it’s most basic step so that the data can be progressed through the algorithm one iteration at a time. Some of the algorithms, like IDS, appear to run much slower than they would in practice due to showing every single step of the DFS algorithm.
